Installing Fossil

This page describes how to build and install Fossil. The whole process is designed to be very easy.

1.0 Obtaining The Source Code

Fossil is self-hosting, so you can obtain a ZIP archive containing a snapshot of the latest version directly from fossil's own fossil repository. Follow these steps:

  1. Pointer your webbrowser at http://fossil-scm.hwaci.com/fossil/login.

  2. Log in as anonymous. The password is shown on screen. The reason for requiring this login is to prevent help spiders from walking the walking the entire website, downloading ZIP archives of every historical version, and thereby soaking up all our bandwidth.

  3. Click on the Timeline or Leaves link at the top of the page.

  4. Select a version of of fossil you want to download. Click on its link. Note that you must successfully log in as "anonymous" in step 1 above in order to see the link to the detailed version information.

  5. On the version information page, click on the "Zip Archive" link. This build a ZIP archive of the complete source code and return it to your browser.

2.0 Compiling

Follow these stops to compile:

  1. Create a directory to hold the source code. Then unzip the ZIP archive you downloaded into that directory. You should be in the top-level folder of that directory

  2. (Optional:) Edit the Makefile to set it up like you want. You probably do not need to do anything. Do not be intimidated: There are only 5 variables in the makefile that can be changed. The whole Makefile is only a few dozen lines long and most of those lines are comments.

  3. Type "make"

3.0 Installing

  1. The finished binary is named "fossil". Put this binary in a directory that is somewhere on your PATH environment variable. It does not matter where.

  2. (Optional:) To uninstall, just delete the binary.
